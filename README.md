# [MVIShopping](https://github.com/Elizav555/MVIShopping) 
This app was my first attempt to work with RxRedux

## Features
- Google one tap signIn
- Edit products list
- Add product to cart
- Sort lists

## Stack
- Firebase Auth
- Firestore
- RxRedux
- RxJava
- Coroutines
- ViewPager2
- Tabs
- Hilt
- MVI
- Clean Architecture
